# Building-Design
AutoCAD Engineering Design Portfolio
Bangladesh Agricultural University (BAU) - Department of FSEE
This repository contains my technical drawings and structural layouts developed during the AutoCAD Engineering Design course under the instruction of Abdullah Al Mahin (Lecturer, BAU).

🛠️ Technical Competencies
Drafting: 2D Layouts, Floor Plans, and Front Elevations.

Structural: Cross-sections & Long-sections of Beams, Columns, and Foundations.

Workflow: Layer Management, Block Attributes, and Custom UCS.

📂 Project Highlights
1. Multistoried Building Layout
Description: A complete architectural set including a floor plan, column layout, and foundation plan.

File: /Architectural/Building_FloorPlan.dwg

2. Structural Detailing
Description: Detailed reinforcement drawings for Beam and Column cross-sections.

File: /Structural/Beam_Detailing.pdf

🚀 Practical Application
As an Agricultural Engineering student, I use these CAD skills to bridge the gap between structural design and sustainable agricultural infrastructure, specifically for my projects in the 2026 Food Systems Innovation Challenge.
